# Strings Theory

- String basics: the `string` type stores text and supports length queries, concatenation with `+`, and indexing into characters.
- Substring operations: extract parts of a string with `substr`, search with `find`, and slice or modify sections as needed.
- Comparison and transformation: compare strings directly, convert numbers with `to_string` and `stoi`, and use character operations to normalize data.
- Common patterns: check palindromes by comparing mirrored characters, detect anagrams using frequency counts, and manipulate substrings for parsing tasks.
- Helpers: use `stringstream` for splitting or joining text, and standard algorithms to sort characters or count occurrences.
