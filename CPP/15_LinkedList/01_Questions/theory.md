# Linked List Theory

- Structure: a linked list is a chain of nodes where each node stores a value and a pointer to the next node.
- Traversal: move from the head through each `next` pointer until the end, often to search or count nodes.
- Insertion and deletion: adjust pointers carefully when adding or removing nodes to preserve the list structure.
- Reversal and cycles: reverse by rewiring pointers iteratively or recursively, and detect cycles using slow and fast pointers.
- Use cases: linked lists support efficient insertions and deletions at arbitrary positions compared to arrays.
