# Inheritance Theory

- Inheritance: derive one class from another to reuse and extend behavior, modeling an "is-a" relationship.
- Single inheritance: one derived class extends one base class.
- Multilevel inheritance: a chain of classes where each class derives from the previous one.
- Hierarchical inheritance: multiple derived classes inherit from the same base class.
- Visibility: the access specifier in inheritance affects how base members are inherited and accessed by derived classes.
