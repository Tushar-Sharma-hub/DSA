# Stack Theory

- LIFO behavior: a stack processes the last inserted element first.
- Push/pop: add items with `push` and remove the top item with `pop`.
- Top/peek: access the current element at the top without removing it.
- Use cases: stacks are used for function call management, expression evaluation, parentheses matching, and DFS traversal.
- Implementation: stacks can be implemented using arrays, vectors, or the STL `stack` container.
