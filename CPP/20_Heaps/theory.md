# Heap Theory

- Heap structure: a complete binary tree where each parent is ordered relative to its children.
- Min-heap: the smallest element is at the root; max-heap: the largest element is at the root.
- Priority queue: use a heap to efficiently access or remove the current highest-priority element.
- Operations: build the heap, push new items, pop the root, and restore the heap property with heapify.
- Use cases: scheduling tasks, finding top-k elements, and implementing heap sort.
