# Searching Theory

- Linear search: examine each element until a match is found; simple and works on unsorted collections.
- Binary search: on sorted arrays, repeatedly compare the target to the midpoint and narrow the search range by half.
- Two-pointer search: use a left and right pointer or sliding window to solve pair-sum and interval problems efficiently.
- Search pattern: identify whether data is sorted or can be transformed, then choose between linear and binary-style search strategies.
- Preconditions: binary search requires sorted input, while linear and two-pointer searches can often handle unsorted arrays with additional logic.
