# Sorting Theory

- Purpose: sorting arranges elements into order so searching, comparison, and ordering tasks become easier.
- Bubble sort: repeatedly swap adjacent out-of-order elements, pushing larger values toward the end in each pass.
- Insertion sort: build a sorted prefix by inserting each new element into its proper position within the already-sorted section.
- Selection sort: repeatedly select the smallest remaining element and place it at the next sorted position.
- Complexity ideas: simple sorts often have `O(n^2)` performance, which is acceptable for small arrays but slower for large input sizes.
