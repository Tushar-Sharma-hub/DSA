# Binary Tree Theory

- Structure: each node has a value and two possible children: left and right.
- Traversals: preorder visits root before children, inorder visits left-root-right, postorder visits children before root, and level order visits nodes by depth.
- Recursive patterns: tree problems often solve left and right subtrees recursively and combine results at the root.
- Properties: compute tree height, count nodes, identify leaves, and assess balanced structure.
- Use cases: binary trees model hierarchical data, decision processes, and recursive relationships.
