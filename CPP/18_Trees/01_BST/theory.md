# Binary Search Tree Theory

- BST property: left subtree values are smaller than the node and right subtree values are larger.
- Search: compare the target with the current node and recursively move left or right.
- Insert: place a new value in the correct subtree while preserving the BST ordering property.
- Delete: remove a node by replacing it with a suitable successor or predecessor and reconnecting children.
- Traversal: inorder traversal of a BST yields values in sorted order.
