# 2D Arrays Theory

- Representation: 2D arrays represent grids or matrices using nested loops where the outer loop iterates rows and the inner loop iterates columns.
- Initialization: use fixed-size arrays like `int a[r][c]` or dynamic containers such as `vector<vector<int>>` for variable dimensions.
- Traversal: visit every element with nested loops, often applying row-wise or column-wise logic for sums, comparisons, or transformations.
- Matrix patterns: common operations include transpose, rotating the matrix, traversing diagonals, generating spiral order, and wave traversal.
- Problem mapping: many grid problems model adjacency implicitly between cells, so 2D array patterns are used for connectivity, boundary checks, and neighbor exploration.
